# Expense Tracker 🧾

A simple command-line based Expense Tracker built using Python.  
This project helps users add expenses, store them in a CSV file, and view a complete spending summary based on categories and budget.

The main purpose of this project was to practice Python fundamentals like:
- Functions
- Loops
- File Handling
- Lists & Dictionaries
- Object-Oriented Programming (OOP)
- User Input Handling
- Match Case Statements

---

## Features 🚀

- Add new expenses
- Select expense categories
- Save expenses into a CSV file
- View category-wise expense summary
- Calculate total spending
- Track remaining budget
- Simple menu-driven interface

---

## Technologies Used 🛠️

- Python
- CSV File Handling
- OOP Concepts
